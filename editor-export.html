Los​‍​‌‍​‍‌ módulos son programas que amplían las funcionalidades de un servidor web como Apache, permitiendo cargar solo las características necesarias y haciendo el servidor más ligero.

Para ver los módulos instalados en Apache, se utiliza el comando

apache2ctl -M

en la terminal.

Módulos de Apache para páginas dinámicas

Los módulos en un servidor web se usan para añadir funcionalidades y capacidades.

En el contexto de las páginas dinámicas, estos módulos se encargan de procesar solicitudes, manejar contenido y comunicarse con lenguajes de programación como PHP.

  • Manejo de contenido:mod_dir: Sirve directorios.mod_mime: Determina el tipo de contenido a enviar al cliente (ej. HTML, imagen, etc.).mod_deflate: Comprime el contenido antes de enviarlo al cliente para reducir el tiempo de carga.
  • mod_dir: Sirve directorios.
  • mod_mime: Determina el tipo de contenido a enviar al cliente (ej. HTML, imagen, etc.).
  • mod_deflate: Comprime el contenido antes de enviarlo al cliente para reducir el tiempo de carga.
  • Comunicación con lenguajes de programación:mod_php: Permite ejecutar scripts PHP.mod_wsgi: Facilita la comunicación con aplicaciones Python.mod_proxy: Permite actuar como proxy inverso para otros servidores.
  • mod_php: Permite ejecutar scripts PHP.
  • mod_wsgi: Facilita la comunicación con aplicaciones Python.
  • mod_proxy: Permite actuar como proxy inverso para otros servidores.
  • Seguridad:mod_ssl: Habilita la encriptación SSL/TLS para conexiones seguras (HTTPS).mod_authn_core: Proporciona la funcionalidad de autenticación básica.
  • mod_ssl: Habilita la encriptación SSL/TLS para conexiones seguras (HTTPS).
  • mod_authn_core: Proporciona la funcionalidad de autenticación básica.

Cómo ver los módulos instalados en Apache

Para listar los módulos que tiene instalado Apache, usa la siguiente instrucción en la terminal del servidor:

  • Abre la terminal del servidor.
  • Ejecuta el comando:

Code

apache2ctl -M

httpd -M

(dependiendo de la distribución de Linux).

  • Esto mostrará una lista de todos los módulos que están actualmente activos en el servidor ​‍​‌‍​‍‌Apache.
